Como criar seu primeiro programa "Hello World!" em Linguagem C

Este guia vai te ajudar a escrever e executar seu primeiro programa em C, que exibe a mensagem "Hello World!" na tela.

Passo a Passo

  1. Instale um compilador: Você precisará de um compilador C. Algumas opções populares incluem:
    • GCC (GNU Compiler Collection)
    • Clang
    • Microsoft Visual C++
  2. Crie um arquivo fonte: Abra seu editor de texto favorito e crie um novo arquivo chamado hello.c.
  3. Escreva o código: Insira o seguinte código no arquivo hello.c: 
    #include <stdio.h>

int main() {
  printf("Hello, World!\\n");
  return 0;
}
  4. Compile o programa: Abra o terminal ou prompt de comando e navegue até o diretório onde o arquivo hello.c está salvo. Execute o comando: gcc hello.c -o hello
  5. Execute o programa: Após a compilação, execute o programa gerado com o comando: ./helloVocê deve ver a mensagem:Hello, World!

Dica: Se estiver usando Windows, o comando para executar pode ser apenas hello (sem o ./).

Conclusão

Parabéns! Você escreveu e executou seu primeiro programa em C. Agora você pode explorar mais e criar programas mais complexos!

Para mais informações sobre programação em C, consulte a documentação oficial ou tutoriais online.
