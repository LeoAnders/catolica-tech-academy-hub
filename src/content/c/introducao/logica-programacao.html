O que é lógica de programação?

Lógica de programação é o caminho do pensamento que usamos para resolver problemas de forma clara e organizada. Antes de escrever código, é importante pensar em como resolver o problema. A lógica de programação te ajuda a planejar cada passo.

Exemplo simples:

Se você quer acender as luzes da sua casa, você segue uma sequência:

  1. Levanta do sofá.
  2. Vai até o interruptor.
  3. Aperta o botão.

Programar funciona da mesma maneira. A lógica organiza o passo a passo antes de você escrever o código.
