Estruturas de Repetição em C

Em C, as estruturas de repetição permitem executar um bloco de código várias vezes, facilitando a realização de tarefas repetitivas. As principais estruturas de repetição em C são o for, o while e o do-while.

Funções Principais:

  1. for: O loop for é ideal para quando o número de iterações é conhecido antes da execução. Ele é composto por três partes: inicialização, condição e incremento/atualização.

    Sintaxe:

    
for (inicialização; condição; incremento) {
  // bloco de código
}

    Exemplo:

    
#include 

int main() {
  int i;
  for (i = 0; i < 5; i++) {
      printf("Iteração %d\n", i);
  }
  return 0;
}

    Saída:

    
Iteração 0
Iteração 1
Iteração 2
Iteração 3
Iteração 4
  2. while: O loop while é usado quando o número de iterações não é conhecido e depende de uma condição ser verdadeira. O bloco de código é executado enquanto a condição for verdadeira.

    Sintaxe:

    
while (condição) {
  // bloco de código
}

    Exemplo:

    
#include 

int main() {
  int i = 0;
  while (i < 5) {
      printf("Iteração %d\n", i);
      i++;
  }
  return 0;
}

    Saída:

    
Iteração 0
Iteração 1
Iteração 2
Iteração 3
Iteração 4
  3. do-while: O loop do-while garante que o bloco de código seja executado pelo menos uma vez antes da verificação da condição. É útil quando você precisa garantir a execução inicial.

    Sintaxe:

    
do {
  // bloco de código
} while (condição);

    Exemplo:

    
#include 

int main() {
  int i = 0;
  do {
      printf("Iteração %d\n", i);
      i++;
  } while (i < 5);
  return 0;
}

    Saída:

    
Iteração 0
Iteração 1
Iteração 2
Iteração 3
Iteração 4

Estruturas de Repetição Aninhadas:

Você também pode aninhar estruturas de repetição para criar loops dentro de loops.

Exemplo:


#include 

int main() {
  int i, j;
  for (i = 0; i < 3; i++) {
      for (j = 0; j < 3; j++) {
          printf("i = %d, j = %d\n", i, j);
      }
  }
  return 0;
}

Saída:


i = 0, j = 0
i = 0, j = 1
i = 0, j = 2
i = 1, j = 0
i = 1, j = 1
i = 1, j = 2
i = 2, j = 0 
i = 2, j = 1
i = 2, j = 2

Dicas Importantes

  • Controle de Fluxo: Use break para sair de um loop e continue para pular para a próxima iteração.
  • Evite Loops Infinitos: Certifique-se de que a condição de saída do loop seja alcançável para evitar loops infinitos.

Veja mais sobre break e continue em nosso tópico exclusivo sobre Controle de Loops.


#include 

int main() {
  for (int i = 0; i < 5; i++) {
      if (i % 2 == 0) continue; // Pula iterações com números pares
      printf("i = %d\n", i);
  }
  return 0;
}
