Variáveis e Tipos de Dados em C

As variáveis são espaços na memória usados para armazenar valores temporários durante a execução de um programa. Para usar uma variável, é necessário declará-la especificando seu tipo de dado.

Declaração de Variáveis

Para declarar uma variável em C, você deve definir o tipo de dado que ela armazenará, seguido do nome da variável. Exemplo: 


    int idade;
    float altura;
    char inicial;

No exemplo acima:

  • idade é uma variável do tipo int (número inteiro),
  • altura é do tipo float (número com ponto flutuante),
  • inicial é do tipo char (caractere único).

Tipos de Dados Primitivos em C

Em C, os principais tipos de dados primitivos são:

  1. int (inteiros):

    Armazena números inteiros, positivos ou negativos, sem decimais.

    int idade = 25;
  2. float (números de ponto flutuante):

    Usado para números decimais com menor precisão.

    float altura = 1.75;
  3. double (números de ponto flutuante com maior precisão):

    Similar ao float, mas com maior capacidade de precisão.

    double salario = 2500.50;
  4. char (caracteres):

    Armazena um único caractere.

    char inicial = 'G';

Operações Básicas

Operações Aritméticas

Em C, você pode realizar operações matemáticas básicas, como:

  • Adição (+): 
    int resultado = 5 + 3;  // resultado será 8
  • Subtração (-): 
    int resultado = 10 - 4;  // resultado será 6
  • Multiplicação (*): 
    int resultado = 7 * 3;  // resultado será 21
  • Divisão (/): 
    float resultado = 10 / 4;  // resultado será 2.5
  • Módulo (%) (resto da divisão): 
    int resultado = 10 % 3;  // resultado será 1

Operações de Incremento e Decremento

  • Incremento (++): aumenta o valor de uma variável em 1. 
    int x = 5;
x++;  // x agora é 6
  • Decremento (--): diminui o valor de uma variável em 1. 
    int y = 5;
y--;  // y agora é 4
