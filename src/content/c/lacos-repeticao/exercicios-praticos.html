Exercícios Práticos: Laços de Repetição e Controle de Fluxo em C

Para consolidar o conhecimento sobre laços de repetição e o uso das instruções break e continue, aqui estão alguns exercícios práticos com suas saídas esperadas e dicas para ajudá-lo a resolver cada um deles.

Exercício 1: Impressão de Números Pares e Ímpares

Objetivo: Imprimir os números de 1 a 20, indicando se cada número é par ou ímpar.

Saída Esperada:

2 é par
4 é par
6 é par
8 é par
10 é par
12 é par
14 é par
16 é par
18 é par
20 é par

Dicas:

  • Use um loop for para iterar de 1 a 20.
  • Utilize a instrução continue para pular a impressão de números ímpares se o objetivo é mostrar apenas números pares.

Exercício 2: Tabuada de um Número

Objetivo: Calcular e imprimir a tabuada de um número fornecido pelo usuário, de 1 a 10.

Saída Esperada para o número 7:

7 x 1 = 7
7 x 2 = 14
7 x 3 = 21
7 x 4 = 28
7 x 5 = 35
7 x 6 = 42
7 x 7 = 49
7 x 8 = 56
7 x 9 = 63
7 x 10 = 70

Dicas:

  • Use scanf para ler o número fornecido pelo usuário.
  • Utilize um loop for para iterar de 1 a 10 e calcular a multiplicação.

Exercício 3: Soma de Números Até um Valor

Objetivo: Calcular a soma de todos os números inteiros positivos menores que um número fornecido pelo usuário, que seja múltiplo de 5. Use break para sair do loop se um número não for múltiplo de 5.

Saída Esperada para o limite 15:

A soma dos múltiplos de 5 é 15

Saída Esperada para o limite 22:

A soma dos múltiplos de 5 é 15

Dicas:

  • Utilize um loop while para iterar até o limite fornecido.
  • Use a instrução continue para pular números que não são múltiplos de 5.
  • Some apenas os números que atendem à condição e exiba o resultado ao final.

Exercício 4: Fatorial de um Número com Validação

Objetivo: Calcular o fatorial de um número positivo fornecido pelo usuário. Se o número for negativo, exibir uma mensagem de erro.

Saída Esperada para o número 5:

O fatorial de 5 é 120

Saída Esperada para o número -3:

Número inválido. O número deve ser positivo.

Dicas:

  • Use um loop do-while para garantir que o usuário insira um número positivo.
  • Utilize um loop for para calcular o fatorial se o número for válido.
  • Use break para sair do loop se o número for negativo e exiba uma mensagem de erro.

Exercício 5: Padrão de Triângulo com Estrelas

Objetivo: Imprimir um padrão de triângulo de estrelas. O usuário deve fornecer o número de linhas para o triângulo.

Saída Esperada para 4 linhas:

*
**
***
****

Dicas:

  • Use um loop for para controlar o número de linhas.
  • Utilize um loop interno para imprimir o número adequado de estrelas em cada linha.

Exercício 6: Contagem Regressiva com break

Objetivo: Contar regressivamente de 10 a 1, mas interromper a contagem se encontrar o número 5.

Saída Esperada:

10
9
8
7
6

Dicas:

  • Use um loop for para a contagem regressiva.
  • Use a instrução break para sair do loop quando o número 5 for alcançado.

Dicas Gerais para os Exercícios:

  • Teste os Resultados: Compare suas saídas com as esperadas para garantir que o comportamento dos seus programas esteja correto.
  • Experimente Modificar: Tente ajustar os parâmetros ou adicionar novas funcionalidades para explorar mais sobre o controle de loops e fluxo.

Nota:

Esses exercícios práticos ajudarão a reforçar seu entendimento sobre laços de repetição e o uso das instruções break e continue em C. Boa prática!
