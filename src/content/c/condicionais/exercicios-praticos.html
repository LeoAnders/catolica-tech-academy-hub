Exercícios Práticos

Aqui estão alguns exercícios práticos para solidificar o entendimento de condicionais em C

  1. Verifique se um número é par ou ímpar

    Peça ao usuário um número e use uma estrutura condicional para verificar se ele é divisível por 2.

    Código

    #include <stdio.h>

int main() {

    int numero;
    printf("Escreva um numero: ");
    scanf("%d", &numero);
            
    if (numero % 2 == 0) {
       printf("O numero e par.\n");
    } else {
       printf("O numero e impar.\n");
    }

}

    Saída:

    Escreva um numero: ...

//se for par
O numero e par
//se for impar
O numero e impar

    2. Dicas

    • Sempre teste com números diferentes.
    • Explique partes importantes com comentários para um melhor entendimento.

  2. Determine o maior entre três números

    Crie um programa que recebe três números e mostra o maior deles.

    Código

    #include <stdio.h>

int main() {

    int a, b, c;
    printf("Digite três números: ");
    scanf("%d %d %d", &a, &b, &c);
    
    if (a >= b && a >= c) {
      printf("O maior número é %d\n", a);
    } else if (b >= a && b >= c) {
      printf("O maior número é %d\n", b);
    } else {
      printf("O maior número é %d\n", c);
    }

}
