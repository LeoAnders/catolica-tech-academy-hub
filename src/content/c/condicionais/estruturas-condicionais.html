Estruturas Condicionais

A importância das Estruturas Condicionais em C

As estruturas condicionais em C são fundamentais para controlar o fluxo de um programa com base em diferentes condições. O comando if executa um bloco de código apenas se uma condição específica for verdadeira. O else é utilizado para tratar o caso onde a condição do if é falsa. Já o else if permite testar várias condições adicionais.

A Declaração if

A declaração if é usada para especificar um bloco de código que será executado se uma condição for verdadeira.

Sintaxe:


if (condition) {
    // bloco de código a ser executado se a condição for verdadeira
}

Note que if está em letras minúsculas. Letras maiúsculas (If ou IF) gerarão um erro.

No exemplo abaixo, testamos dois valores para descobrir se 20 é maior que 18. Se a condição for verdadeira, imprime um texto:

Exemplo:


if (20 > 18) {
    printf("20 é maior que 18");
}

Também podemos testar variáveis:

Exemplo:


int x = 20;
int y = 18;

if (x > y) {
    printf("x é maior que y");
}

No exemplo acima, usamos duas variáveis, x e y, para testar se x é maior que y (usando o operador >). Como x é 20 e y é 18, e sabemos que 20 é maior que 18, imprimimos na tela que "x é maior que y".

A Declaração else

A declaração else é usada para especificar um bloco de código que será executado se a condição for falsa.

Sintaxe:


if (condition) {
    // bloco de código a ser executado se a condição for verdadeira

} else {
    // bloco de código a ser executado se a condição for falsa
}

Exemplo:


int time = 20;

if (time < 18) {
    printf("Bom dia.");

} else {
    printf("Boa noite.");
}

// Saída: "Boa noite."

No exemplo acima, time (20) é maior que 18, então a condição é falsa. Por isso, o código segue para o bloco else e imprime "Boa noite" na tela. Se time fosse menor que 18, o programa imprimiria "Bom dia".

A Declaração else if

A declaração else if é usada para especificar uma nova condição se a primeira for falsa.

Sintaxe:


if (condition1) {
    // bloco de código a ser executado se a condition1 for verdadeira
} else if (condition2) {
    // bloco de código a ser executado se a condition1 for falsa e a condition2 for verdadeira
} else {
    // bloco de código a ser executado se condition1 e condition2 forem falsas
}

Exemplo:


int time = 22;

if (time < 10) {
    printf("Bom dia.");
} else if (time < 20) {
    printf("Boa tarde.");
} else {
    printf("Boa noite.");
}
// Saída: "Boa noite."

No exemplo acima, time (22) é maior que 10, então a primeira condição é falsa. A próxima condição, no else if, também é falsa, então seguimos para o bloco else, onde imprimimos "Boa noite" na tela. No entanto, se time fosse 14, o programa imprimiria "Boa tarde".
