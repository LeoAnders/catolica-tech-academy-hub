Exemplos de Laços de Repetição em C

Os laços de repetição em C permitem executar um bloco de código várias vezes, com base em condições ou contadores. Abaixo estão exemplos práticos dos diferentes tipos de laços de repetição: for, while e do-while.

1. Exemplo com for

O laço for é útil quando o número de iterações é conhecido de antemão. Ele permite a inicialização, condição e incremento em uma única linha.

Exemplo 1: Imprimir Números de 1 a 10

#include <stdio.h>

int main() {
    int i;
    for (i = 1; i <= 10; i++) {
        printf("%d\n", i);
    }
    return 0;
}

Exemplo 2: Soma dos Números de 1 a 5

#include <stdio.h>

int main() {
    int i;
    int soma = 0;
    for (i = 1; i <= 5; i++) {
        soma += i;
    }
    printf("A soma é %d\n", soma);
    return 0;
}

Saída:

A soma é 15

2. Exemplo com while

O laço while é usado quando o número de iterações não é conhecido de antemão e depende de uma condição.

Exemplo 1: Imprimir Números de 1 a 10

#include <stdio.h>

int main() {
    int i = 1;
    while (i <= 10) {
        printf("%d\n", i);
        i++;
    }
    return 0;
}

1
2
3
4
5

Exemplo 2: Encontrar o Fatorial de um Número

#include <stdio.h>

int main() {
    int n = 5;
    int fatorial = 1;
    while (n > 0) {
        fatorial *= n;
        n--;
    }
    printf("O fatorial é %d\n", fatorial);
    return 0;
}

Saída:

O fatorial é 120

3. Exemplo com do-while

O laço do-while garante que o bloco de código seja executado pelo menos uma vez antes da verificação da condição.

Exemplo 1: Imprimir Números de 1 a 10

#include <stdio.h>

int main() {
    int i = 1;
    do {
        printf("%d\n", i);
        i++;
    } while (i <= 10);
    return 0;
}

Exemplo 2: Contar de 10 a 1

#include <stdio.h>

int main() {
    int i = 10;
    do {
        printf("%d\n", i);
        i--;

    return 0;
}

4. Exemplos de Laços Aninhados

Laços de repetição podem ser aninhados para criar padrões complexos ou realizar operações em matrizes.

Exemplo 1: Imprimir uma Tabela de Multiplicação

#include <stdio.h>

int main() {
    int i, j;
    for (i = 1; i <= 10; i++) {
        for (j = 1; j <= 10; j++) {
            printf("%4d", i * j);
        }
        printf("\n");
    }
    return 0;
}

Exemplo 2: Criar um Padrão de Estrela

#include <stdio.h>

int main() {
    int i, j;
    int n = 5; // Número de linhas
    for (i = 1; i <= n; i++) {
        for (j = 1; j <= i; j++) {
            printf("*");
        }
        printf("\n");
    }
    return 0;
}

Saída:

*
**
***
****
*****

Nota:

Com estes exemplos, você tem uma compreensão prática de como usar laços de repetição em C para realizar várias tarefas. Esses laços são ferramentas poderosas para automatizar e simplificar a execução repetitiva de blocos de código.
