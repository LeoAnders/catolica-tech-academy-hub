Comandos de Entrada e Saída em C

A linguagem C oferece diversas funções para realizar operações de entrada (input) e saída (output), permitindo que você leia e exiba dados. As funções mais utilizadas estão na biblioteca padrão stdio.h.

Funções Principais:

1. printf(): Usada para exibir informações na tela (saída).

printf("formato", variáveis)

        
        
        #include <stdio.h>
        
        int main() {

            int idade = 25;

            printf("Minha idade é %d anos\n", idade);

            return 0;
        }

Saída:

Minha idade é 25 anos

2. scanf(): Usada para ler entradas do teclado (entrada).

scanf("formato", &variável);

        
        #include <stdio.h>

        int main() {

            int idade;
            printf("Digite sua idade: ");
            scanf("%d", &idade);

            printf("Você tem %d anos\n", idade);

            return 0;
        }

Saída:

Você tem 25 anos

Formatos de Conversão Comuns:

Formato Descrição
%d Inteiros (int)
%f Números de ponto flutuante
%c Caracteres (char)
%s Strings (sequência de caracteres)

Tratando Entradas e Saídas com Strings:

Exibindo Strings com printf():

printf("Olá, %s!\n", "Gabriel");

Saída:

Olá, Gabriel!

Lendo Strings com scanf():

        
            #include <stdio.h>

            int main() {
            char nome[50];
            printf("Digite seu nome: ");
            scanf("%s", nome);
            printf("Olá, %s!\n", nome);
            return 0;
            }

Entrada:

Gabriel

Saída:

Olá, Gabriel!

Dicas Importantes:

  • Sempre use o & (endereço) antes de variáveis ao usar scanf(), exceto para strings.
  • O printf() e o scanf() são altamente customizáveis. Além dos formatos básicos, você pode adicionar controle de precisão e largura, por exemplo: %5d alinha o número em 5 espaços.

Funções Adicionais:

1. gets() (não recomendada): Lê uma linha inteira de texto (string), mas é insegura pois não verifica limites de buffer.

        

            #include <stdio.h>

                int main() {
                char nome[50];
                gets(nome);

                printf("Nome: %s\n", nome);

                return 0;
            }

2. fgets(): Substitui o gets() de forma mais segura, especificando o número máximo de caracteres a serem lidos.

        
        #include <stdio.h>

            int main() {
            char nome[50];
            printf("Digite seu nome: ");
            fgets(nome, 50, stdin);
            printf("Nome: %s", nome);
            return 0;
        }

Saída:

Nome: Gabriel
