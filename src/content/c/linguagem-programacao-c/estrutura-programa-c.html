Estrutura básica de um programa em C

A estrutura básica de um programa em C segue uma organização específica e inclui os seguintes elementos principais:

Bibliotecas (Headers)

No início do programa, as bibliotecas necessárias são incluídas com o comando #include. A mais comum é a stdio.h, que fornece funções de entrada e saída como printf e scanf.


#include <stdio.h>

Função principal (main)

Todo programa em C deve ter uma função principal, int main(), que é o ponto de partida da execução. Pode retornar um valor inteiro para indicar o sucesso (0) ou falha de execução.


int main() { 
  // código do programa
  return 0;
}

Declaração de variáveis

As variáveis devem ser declaradas antes de serem usadas no programa.


int a, b;

Instruções

O corpo da função contém as instruções que implementam a lógica do programa, como operações matemáticas, chamadas de funções, loops, etc.


printf("Hello, World!");

Retorno

A função main normalmente termina com return 0;, que indica que o programa foi concluído com sucesso.

Exemplo básico de um programa em C


#include <stdio.h>

int main() {
  printf("Hello, World!\n");
  return 0;
}

Nota

Este código imprime "Hello, World!" no console e retorna 0, indicando que foi executado com sucesso.
