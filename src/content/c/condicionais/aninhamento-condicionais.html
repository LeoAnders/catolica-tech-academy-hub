Alinhamento de Condicionais e Estruturas de Decisão

Importância do Alinhamento

O alinhamento adequado do código é crucial para a legibilidade e manutenção. Em C, o uso de indentação torna claro onde começam e terminam os blocos de código dentro de cada if, else if, ou else. Evitar aninhamentos desnecessários e manter a lógica bem organizada também é importante.

Exemplo de Alinhamento de Condicionais

Aqui está um exemplo de como o alinhamento adequado ajuda a legibilidade do código:

Exemplo


int score = 85;

if (score >= 90) {
    printf("Nota A\n");
} else if (score >= 80) {
    printf("Nota B\n");
} else {
    printf("Nota C\n");
}

Nota:

  • Este código avalia a variável `score`, que armazena uma pontuação. Se a pontuação for 90 ou mais, imprime "Nota A". Se estiver entre 80 e 89, imprime "Nota B". Para pontuações abaixo de 80, imprime "Nota C". Assim, o programa classifica o desempenho em três categorias.

Organização do Código

O uso correto de if, else if e else em conjunto com a indentação apropriada garante que o código seja bem organizado e fácil de manter, além de evitar aninhamentos desnecessários que podem dificultar a compreensão.
