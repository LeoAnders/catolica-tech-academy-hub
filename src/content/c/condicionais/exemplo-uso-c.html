Exemplo de Uso em C

Um exemplo simples de uso das estruturas condicionais com operadores relacionais e lógicos em C


int temperature = 30;

if (temperature > 35) {
    printf("E um dia quente.\n");
} else if (temperature >= 25 && temperature <= 35) {
    printf("E um dia agradavel.\n");
} else {
    printf("E um dia frio.\n");
}

Dicas

  • Antes de codificar, escreva uma lista das condições e resultados esperados. Isso ajuda a organizar o pensamento.
  • Use a indentação correta para tornar o código mais legível. Um código bem formatado facilita a identificação de erros.

Explicação

Neste exemplo, o programa decide que mensagem exibir com base na temperatura informada.
