Operadores Relacionais e Lógicos

Operadores Relacionais

Os operadores relacionais em C são usados para comparar dois valores ou expressões:

  • == (igual a)
  • != (diferente de)
  • > (maior que)
  • < (menor que)
  • >= (maior ou igual a)
  • <= (menor ou igual a)

Operadores Lógicos

Os operadores lógicos combinam expressões booleanas:

  • && (E lógico): retorna verdadeiro se ambas as condições forem verdadeiras.
  • || (OU lógico): retorna verdadeiro se **pelo menos uma** das condições for verdadeira.
  • ! (NÃO lógico): inverte o valor de uma condição (de verdadeiro para falso, ou vice-versa).

Exemplo de Operadores Relacionais e Lógicos


int x = 5, y = 10;

if (x < y && y < 20) {
    printf("A condicao e verdadeira.\n");
}

Exemplo de OU Lógico


int x = 3;

if (x < 5 || x < 4) {
    printf("Uma das condicoes e verdadeira.\n");
} else {
    printf("Nenhuma das condicoes e verdadeira.\n");
}

Operadores de Atribuição Compostos

Os operadores de atribuição compostos permitem simplificar expressões de atribuição:

  • += (adição): x += 3 é equivalente a x = x + 3
  • -= (subtração): x -= 3 é equivalente a x = x - 3
  • *= (multiplicação): x *= 3 é equivalente a x = x * 3
  • /= (divisão): x /= 3 é equivalente a x = x / 3
  • %= (módulo): x %= 3 é equivalente a x = x % 3

Exemplo de Uso dos Operadores de Atribuição Compostos


int x = 10;

x += 3;  // x agora é 13
x -= 2;  // x agora é 11
x *= 4;  // x agora é 44
x /= 4;  // x agora é 11
x %= 3;  // x agora é 2
