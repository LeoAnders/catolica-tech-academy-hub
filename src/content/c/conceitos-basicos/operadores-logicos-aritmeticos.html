Operadores Aritméticos

Os operadores aritméticos são usados para realizar operações matemáticas em variáveis e valores. Aqui estão os principais operadores aritméticos em C:

  • + : Adição
  • - : Subtração
  • * : Multiplicação
  • / : Divisão
  • % : Módulo (resto da divisão)

Exemplo:


#include <stdio.h>

int main() {
    int a = 10, b = 3;
    int soma = a + b;
    int subtracao = a - b;
    int multiplicacao = a * b;
    int divisao = a / b;
    int modulo = a % b;

    printf("Soma: %d\\n", soma);
    printf("Subtração: %d\\n", subtracao);
    printf("Multiplicação: %d\\n", multiplicacao);
    printf("Divisão: %d\\n", divisao);
    printf("Módulo: %d\\n", modulo);

    return 0;
}

Operadores Lógicos

Os operadores lógicos são usados para combinar expressões booleanas. Aqui estão os principais operadores lógicos em C:

  • && : E lógico (AND)
  • || : OU lógico (OR)
  • ! : NÃO lógico (NOT)

Exemplo:


#include <stdio.h>

int main() {
    int x = 5, y = 10;
    int resultado;

    // AND lógico
    resultado = (x < 10 && y > 5);
    printf("Resultado (x < 10 && y > 5): %d\\n", resultado);

    // OR lógico
    resultado = (x > 10 || y > 5);
    printf("Resultado (x > 10 || y > 5): %d\\n", resultado);

    // NOT lógico
    resultado = !(x > 10);
    printf("Resultado (!(x > 10)): %d\\n", resultado);

    return 0;
}
